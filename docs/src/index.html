Overview

MathFu is a C++ math library that developers can use for games. It is optimized for both ARM and x86 hardware. It implements vector, matrix, and quaternion each with basic math operations including *, +, -, /, DotProduct, CrossProduct, Inverse, and Slerp.

MathFu source code is available for download from github.com/google/mathfu.

Discuss MathFu with other developers and users on the mailing list. Report issues on the issues tracker.
